PyCellFIT Web

PyCellFIT Web is a tool to infer forces in epithelial cell layers. Built in Python, it is an implementation of the CellFIT method first introduced by Broadland et al. Starting with a segmented image of a cell layer, tensions along cell membranes and pressures within cells can be computed by approximating cell boundaries as poly-arcs.
